# Product Features

- **Centralized Image Storage**: Provides a secure and reliable repository for Docker images and OCI artifacts, accessible from various environments.
- **Version Control and Tagging**: Supports tagging and versioning of container images, allowing for easy tracking of changes, rollback capabilities, and consistent deployments.
- **Access Control (RBAC)**: Offers robust role-based access control to manage who can push, pull, or delete images within the registry, ensuring data security and integrity.
- **Integration with CI/CD Workflows**: Designed to seamlessly integrate with Continuous Integration/Continuous Delivery pipelines for automated image building, pushing, and deployment.
- **Scalability and Performance**: Engineered for high availability and performance, capable of handling large volumes of image storage and retrieval operations.
